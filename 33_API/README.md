# 🌐 API Practice Projects

This project was built as part of **Day 33** of the
**100 Days of Code: The Complete Python Pro Bootcamp** by Angela Yu.

## 📌 Project Overview

Day 33 focused on learning how to work with **APIs (Application Programming Interfaces)** in Python.

Through a series of exercises and mini-projects, I learned how to send requests to APIs, retrieve data, process JSON responses, and display useful information to users.

These projects demonstrated how Python applications can communicate with external services and use real-time data from the internet.

---

## 🧠 Concepts Practiced

* APIs and HTTP requests
* Working with the `requests` library
* JSON data handling
* API endpoints and parameters
* Response status codes
* Parsing and processing data
* Error handling

## 🚀 Technologies Used

* JSON
