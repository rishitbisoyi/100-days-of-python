# 💰 Tip Calculator

This project was built as part of **Day 2** of the
**100 Days of Code: The Complete Python Pro Bootcamp** by Angela Yu.

## 📌 Project Overview

The Tip Calculator is a simple Python program that helps users calculate how much each person should pay when splitting a bill, including the tip percentage.

The program takes:

* The total bill amount
* Desired tip percentage
* Number of people splitting the bill

It then calculates and displays the final amount each person needs to pay.
