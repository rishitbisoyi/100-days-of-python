# ✊✋✌️ Snake Water Gun / Rock Paper Scissors Game

This project was built as part of **Day 4** of the
**100 Days of Code: The Complete Python Pro Bootcamp** by Angela Yu.

## 📌 Project Overview

This is a command-line game project based on the classic **Snake Water Gun** game.
The program allows the user to play against the computer, where the winner is decided based on predefined game rules.

I also included a previously created **Rock Paper Scissors** version that I built earlier while learning Python basics, as both games follow a similar logic structure.

---

## 🧠 Concepts Practiced

* Conditional statements (`if`, `elif`, `else`)
* Randomization using the `random` module
* User input handling
* Game logic implementation
* Loops and replay functionality
* Comparing user and computer choices

---

## 🚀 Technologies Used

* Python 3

---

## 💡 What I Learned

Through these projects, I practiced building interactive terminal-based games and improved my understanding of program flow and logical decision-making.

I also learned how to generate random computer choices and structure game rules efficiently in Python.

---

## 🎮 Features

* Player vs Computer gameplay
* Random computer moves
* Win/Lose/Draw detection
* Simple and interactive terminal interface
* Includes both Snake Water Gun and Rock Paper Scissors versions
