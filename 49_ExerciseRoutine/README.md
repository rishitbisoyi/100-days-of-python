# 🏋️ Exercise Routine Automation

This project was completed as part of **Day 49** of the  
**100 Days of Code: The Complete Python Pro Bootcamp** by Angela Yu.

## 📌 Project Overview

This project automates the process of logging into a website and completing a daily exercise routine check-in using Selenium WebDriver.

- Selenium WebDriver
- Locating Web Elements
- Form Automation
- User Authentication

## 🚀 Technologies Used

- Python
- Selenium WebDriver

## 💡 What I Learned

Through this project, I learned how to automate login workflows, interact with web pages that require authentication, and perform repetitive tasks using Selenium.
