# 🎨 Hirst Painting using Turtle Graphics

This project was built as part of **Day 18** of the
**100 Days of Code: The Complete Python Pro Bootcamp** by Angela Yu.

## 📌 Project Overview

This project recreates a colorful **Hirst-style dot painting** using Python's Turtle graphics module.

The program uses randomly generated colors and Turtle movements to draw a grid of colored dots inspired by the artwork style of Damien Hirst.

This was also an introduction to working with graphical user interfaces (GUI) in Python using the Turtle module.
