# ✈️ Flight Club

This project was built as part of **Day 40** of the
**100 Days of Code: The Complete Python Pro Bootcamp** by Angela Yu.

## 📌 Project Overview

Flight Club is an extension of the Flight Deal Finder project that enhances the user experience by adding email notifications and customer management.

Users can register with their email addresses, and whenever a flight deal is found below the target price, the application automatically sends email notifications to subscribed users.

This project demonstrates how Python can combine APIs, data management, and email automation to build a practical travel notification system.
