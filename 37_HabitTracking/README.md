# 📊 Habit Tracking Application

This project was built as part of **Day 37** of the
**100 Days of Code: The Complete Python Pro Bootcamp** by Angela Yu.

## 📌 Project Overview

The Habit Tracking Application is a Python project that uses an external habit-tracking API to record and visualize daily habits.

Users can create habits, log their daily progress, update existing records, and manage their tracking data through API requests. This project demonstrates how Python can interact with web services to build productivity and self-improvement tools.

---

## 🧠 Concepts Practiced

* Working with REST APIs
* HTTP methods (`GET`, `POST`, `PUT`, `DELETE`)
* Authentication using API tokens
* Sending JSON data
* Processing API responses
* Automating daily habit tracking

---

## 🚀 Technologies Used

* Python 3
* Requests Library
* Habit Tracking API
* JSON

---

## 💡 What I Learned

Through this project, I learned how to interact with REST APIs using different HTTP methods and authenticate requests securely.

This project also helped me understand how APIs can be used to build practical applications for tracking and managing personal data.

## 🎮 Features

* Create new habits
