# 🔐 Password Generator

This project was built as part of **Day 5** of the
**100 Days of Code: The Complete Python Pro Bootcamp** by Angela Yu.

## 📌 Project Overview

The Password Generator is a Python program that creates secure and randomized passwords based on user preferences.

The user can choose:

* Number of letters
* Number of symbols
* Number of numbers

The program then generates a randomized password using a combination of these characters.

---

## 🧠 Concepts Practiced

* Python lists
* Loops (`for` loops)
* Randomization using the `random` module
* List manipulation
* String concatenation
* User input handling
* Shuffling data

---

## 🚀 Technologies Used

* Python 3

## 🔒 Features

* Generates randomized passwords
* Customizable password length
