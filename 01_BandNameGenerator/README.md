# 🎸 Band Name Generator

This project was built as part of **Day 1** of the
**100 Days of Code: The Complete Python Pro Bootcamp** by Angela Yu.

## 📌 Project Overview

The Band Name Generator is a beginner-friendly Python project that creates a fun band name based on user input.

The program asks the user for:

* The city they grew up in
* The name of their pet

It then combines the answers to generate a simple band name.

---

## 🧠 Concepts Practiced

* Python `print()` statements
* User input with `input()`
* Variables
* String concatenation
* Basic terminal interaction

---

## 🚀 Technologies Used

* Python 3
