# 📈 Stock News Alert

This project was built as part of **Day 36** of the
**100 Days of Code: The Complete Python Pro Bootcamp** by Angela Yu.

## 📌 Project Overview

The Stock News Alert is a Python application that monitors stock price changes and retrieves the latest news related to a company.

The program compares recent stock prices, and if a significant price movement is detected, it fetches relevant news articles and sends them as notifications.

This project demonstrates how multiple APIs can work together to create a useful real-world automation application.
