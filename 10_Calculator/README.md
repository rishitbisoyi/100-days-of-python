# 🧮 Calculator Program

This project was built as part of **Day 10** of the
**100 Days of Code: The Complete Python Pro Bootcamp** by Angela Yu.

## 📌 Project Overview

The Calculator Program is a command-line Python application capable of performing basic mathematical operations.

The user can:

* Add numbers
* Subtract numbers
* Multiply numbers
* Divide numbers

The program also allows continuous calculations by using previous results in future operations.

---

## 🧠 Concepts Practiced

* Python functions
* Function parameters and return values
* Dictionaries with functions
* Loops and conditionals
* Mathematical operations
* Program flow control
* Code organization and reusability

---

## 🚀 Technologies Used

* Python 3

## 💡 What I Learned

Through this project, I learned how functions can be stored and used dynamically with dictionaries to make programs cleaner and more scalable.

This project also helped me better understand reusable code, returning values from functions, and structuring larger Python programs logically.
