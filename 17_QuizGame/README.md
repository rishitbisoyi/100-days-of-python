# ❓ Quiz Game

This project was built as part of **Day 17** of the
**100 Days of Code: The Complete Python Pro Bootcamp** by Angela Yu.

## 📌 Project Overview

The Quiz Game is a command-line Python application that presents multiple-choice style true/false questions to the user and keeps track of the score throughout the game.

The project was built using **Object-Oriented Programming (OOP)** concepts to organize quiz questions and game logic efficiently.

---

## 🧠 Concepts Practiced

* Object-Oriented Programming (OOP)
* Classes and objects
* Constructors (`__init__`)
* Working with multiple Python files/modules
* Lists and dictionaries
* Loops and conditional statements
* Managing program state

---

## 🚀 Technologies Used

* Python 3

This project also helped me understand how data and logic can work together to create interactive applications.
