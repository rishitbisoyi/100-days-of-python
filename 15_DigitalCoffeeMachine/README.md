# ☕ Digital Coffee Machine

This project was built as part of **Day 15** of the
**100 Days of Code: The Complete Python Pro Bootcamp** by Angela Yu.

## 📌 Project Overview

The Digital Coffee Machine is a command-line Python application that simulates the working of a coffee vending machine.

Users can:

* Choose different coffee options
* Insert coins
* Receive drinks if sufficient resources and payment are available

The program also tracks available ingredients and handles transactions automatically.
