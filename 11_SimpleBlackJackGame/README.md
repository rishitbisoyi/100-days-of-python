# 🃏 Simple Blackjack Game

This project was built as part of **Day 11** of the
**100 Days of Code: The Complete Python Pro Bootcamp** by Angela Yu.

## 📌 Project Overview

This project is a simplified command-line version of the classic **Blackjack** card game built using Python.

The player competes against the computer (dealer) by drawing cards and trying to get a score as close to 21 as possible without going over.

The game follows basic Blackjack rules, including:

* Card dealing
* Score calculation
* Dealer logic
* Win/Lose/Draw conditions
