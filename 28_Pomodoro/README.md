# 🍅 Pomodoro Timer

This project was built as part of **Day 28** of the
**100 Days of Code: The Complete Python Pro Bootcamp** by Angela Yu.

## 📌 Project Overview

The Pomodoro Timer is a GUI-based productivity application built using **Tkinter**.

The application follows the **Pomodoro Technique**, a time-management method that alternates focused work sessions with short breaks and longer breaks after multiple work sessions.

The timer automatically switches between work and break periods while visually tracking progress.
