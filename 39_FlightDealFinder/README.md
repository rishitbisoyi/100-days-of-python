# ✈️ Flight Deal Finder

This project was built as part of **Day 39** of the
**100 Days of Code: The Complete Python Pro Bootcamp** by Angela Yu.

## 📌 Project Overview

The Flight Deal Finder is a Python application that searches for affordable flight deals using external APIs.

The program retrieves destination data, checks current flight prices, and compares them against predefined target prices. When a flight price drops below the target, it notifies the user about the deal.

This project demonstrates how Python can automate travel deal tracking by integrating multiple APIs.
