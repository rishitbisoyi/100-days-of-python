# 🏝️ Treasure Island Game

This project was built as part of **Day 3** of the
**100 Days of Code: The Complete Python Pro Bootcamp** by Angela Yu.

## 📌 Project Overview

Treasure Island is a text-based adventure game built using Python.
The player makes choices throughout the game to navigate different paths and try to find the hidden treasure.

Depending on the decisions made, the player can either win the game or encounter different game-over scenarios.

---

## 🧠 Concepts Practiced

* Conditional statements (`if`, `elif`, `else`)
* Nested conditions
* User input handling
* String methods (`lower()`)
* Logical thinking and program flow
* Creating interactive terminal games

---

## 🚀 Technologies Used

* Python 3

---

## 💡 What I Learned

Through this project, I practiced using conditional logic to control the flow of a program and create multiple outcomes based on user choices.

This project also helped me understand how decision-making systems work in interactive applications and games.

## 🎮 Features

* Multiple story paths
* Interactive decision-based gameplay
* Different endings based on player choices
