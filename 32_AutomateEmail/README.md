# 📧 Birthday Email Automation

This project was built as part of **Day 32** of the
**100 Days of Code: The Complete Python Pro Bootcamp** by Angela Yu.

## 📌 Project Overview

This project automates the process of sending birthday wishes via email using Python.

The application checks a list of birthdays, compares them with the current date, and automatically sends a personalized birthday email when a match is found.

Through this project, I learned how Python can interact with email servers and use date-based logic to automate real-world tasks.

---

## 🧠 Concepts Practiced

* SMTP (Simple Mail Transfer Protocol)
* Sending emails with Python
* Working with the `datetime` module
* Reading data from CSV files
* Data filtering using Pandas
* Automation workflows
* Exception handling

---

## 🚀 Technologies Used

* Python 3
* SMTP
* Pandas
* Datetime Module

## 💡 What I Learned

Through this project, I learned how to send emails programmatically and how date and time operations can be used to trigger automated actions.

This project also helped me understand how Python can combine data processing and automation to create practical real-world applications.
