# 🇮🇳 Indian States Quiz

This project was built as part of **Day 25** of the
**100 Days of Code: The Complete Python Pro Bootcamp** by Angela Yu.

## 📌 Project Overview

The Indian States Quiz is an interactive Python game where players try to guess the names of Indian states.

Using a map and data from a CSV file, the program checks user answers and displays correctly guessed states on the map. The game continues until the player guesses all states or chooses to exit.

This project combines graphical programming with data handling to create an educational quiz experience.
