# 🖥️ Introduction to Tkinter

This project was completed as part of **Day 27** of the
**100 Days of Code: The Complete Python Pro Bootcamp** by Angela Yu.

## 📌 Project Overview

Day 27 focused on learning the fundamentals of **Tkinter**, Python's standard GUI (Graphical User Interface) library.

During this lesson, I explored how to create graphical applications using windows, labels, buttons, input fields, and event handling.

Several small practice applications and exercises were created to understand how Tkinter works and how user interactions can be handled through a graphical interface.

---

## 🧠 Concepts Practiced

* Tkinter basics
* Creating windows and widgets
* Labels
* Buttons
* Entry fields
* Layout managers (`pack`, `grid`, `place`)
* Event handling
* Function arguments (`*args` and `**kwargs`)

---

## 🚀 Technologies Used

* Python 3
* Tkinter

---

## 💡 What I Learned

Through this day, I learned the foundations of GUI development in Python and how graphical applications differ from traditional command-line programs.

I also learned how widgets interact with user input and how events can trigger functions within an application.
