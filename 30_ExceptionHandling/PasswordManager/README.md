# 🔐 Password Manager

This project was built as part of **Day 29** of the
**100 Days of Code: The Complete Python Pro Bootcamp** by Angela Yu.

## 📌 Project Overview

The Password Manager is a GUI-based Python application built using **Tkinter** that helps users generate, store, and manage login credentials.

Users can enter a website name, email/username, and password. The application can also generate strong random passwords and save the information locally for future use.
