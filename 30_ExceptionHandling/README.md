# ⚠️ Advanced Exception Handling

This project was completed as part of **Day 30** of the
**100 Days of Code: The Complete Python Pro Bootcamp** by Angela Yu.

## 📌 Project Overview

Day 30 focused on improving the Password Manager project by implementing proper exception handling and making the application more robust.

The goal was to anticipate potential errors, handle them gracefully, and provide a better user experience instead of allowing the program to crash unexpectedly.

* `try`, `except`, `else`, and `finally`
* Raising exceptions
* Handling file-related errors
* Input validation
* Defensive programming
* Improving existing applications

## 🚀 Technologies Used

* JSON
* Exception Handling
