# 🌦️ Weather Alert Application

This project was built as part of **Day 35** of the
**100 Days of Code: The Complete Python Pro Bootcamp** by Angela Yu.

## 📌 Project Overview

The Weather Alert Application uses a weather API to retrieve forecast data and determine whether rain is expected.

If rain is predicted within the forecast period, the application sends a notification reminding the user to carry an umbrella. This project demonstrates how APIs and automation can be combined to create practical real-world applications.

---

## 🧠 Concepts Practiced

* Working with REST APIs
* HTTP requests using the `requests` library
* Processing JSON data
* Reading weather forecast data
* Conditional logic
* Automation with notifications
* Environment variables for API keys

---

## 🚀 Technologies Used

* Python 3
* Requests Library
* Weather API
* JSON
