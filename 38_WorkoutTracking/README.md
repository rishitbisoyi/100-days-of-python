# 🏋️ Workout Tracking Application

This project was built as part of **Day 38** of the
**100 Days of Code: The Complete Python Pro Bootcamp** by Angela Yu.

## 📌 Project Overview

The Workout Tracking Application is a Python project that automatically records workout activities by using natural language processing and external APIs.

Users can describe their exercise in plain English, and the application identifies the activities, estimates details such as duration and calories burned, and logs the workout to a spreadsheet for future reference.

This project demonstrates how Python can combine multiple APIs to automate fitness tracking.

---

## 🧠 Concepts Practiced

* Working with REST APIs
* HTTP requests using the `requests` library
* Natural Language Processing (NLP)
* Processing JSON data
* Authentication with API keys
* Logging data to online spreadsheets
* API integration

---

## 🚀 Technologies Used

* Python 3
* Requests Library
* Nutritionix API
* Sheety API
* JSON
