# 🛒 Amazon Price Tracker

This project was completed as part of **Day 47** of the  
**100 Days of Code: The Complete Python Pro Bootcamp** by Angela Yu.

## 📌 Project Overview

This project tracks the price of an Amazon product by scraping its product page. If the price falls below a target value, an email notification is sent to the user.

---

## 🧠 Concepts Practiced

- Web Scraping
- Beautiful Soup
- HTTP Requests
- Sending Emails with SMTP
- Automation

---

## 🚀 Technologies Used

- Python
- Requests
- Beautiful Soup
- SMTP

---

## 💡 What I Learned

Through this project, I learned how to scrape product information from a website, monitor price changes, and automate email notifications based on specific conditions.
