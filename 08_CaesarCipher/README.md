# 🔐 Caesar Cipher

This project was built as part of **Day 8** of the
**100 Days of Code: The Complete Python Pro Bootcamp** by Angela Yu.

## 📌 Project Overview

The Caesar Cipher project is a text encryption and decryption program built using Python.

The program allows users to:

* Encrypt messages
* Decrypt encrypted text
* Choose the shift value used in the cipher

The Caesar Cipher works by shifting letters in the alphabet by a specified number of positions.
