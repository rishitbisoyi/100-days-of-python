# 🐢 Turtle Crossing Game

This project was built as part of **Day 23** of the
**100 Days of Code: The Complete Python Pro Bootcamp** by Angela Yu.

## 📌 Project Overview

The Turtle Crossing Game is a graphical Python game inspired by the classic arcade-style road crossing games.

The player controls a turtle and attempts to cross the road while avoiding moving cars.
As the player progresses, the game difficulty increases with faster-moving traffic.

---

## 🧠 Concepts Practiced

* Object-Oriented Programming (OOP)
* Turtle graphics
* Collision detection
* Game loops and screen updates
* Keyboard event handling
* Managing multiple moving objects
* Difficulty progression logic

---

## 🚀 Technologies Used

* Python 3
* Turtle Module

Through this project, I learned how to manage multiple moving objects dynamically and create progressive difficulty in a game.

This project also helped me improve my understanding of collision detection, object interaction, and organizing larger graphical Python projects.
