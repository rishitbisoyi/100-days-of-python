# 📡 NATO Phonetic Alphabet Converter

This project was built as part of **Day 26** of the
**100 Days of Code: The Complete Python Pro Bootcamp** by Angela Yu.

## 📌 Project Overview

The NATO Phonetic Alphabet Converter is a Python program that converts a user-entered word into its corresponding NATO phonetic alphabet code words.

The program reads data from a CSV file and creates a dictionary that maps each letter of the alphabet to its NATO code word. Users can then enter any word and receive its phonetic representation.

For example:

```text
Input:  RISHIT
Output: ['Romeo', 'India', 'Sierra', 'Hotel', 'India', 'Tango']
```
