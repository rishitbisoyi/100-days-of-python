# 🏓 Pong Game

This project was built as part of **Day 22** of the
**100 Days of Code: The Complete Python Pro Bootcamp** by Angela Yu.

## 📌 Project Overview

The Pong Game is a graphical Python recreation of the classic arcade game **Pong** built using the Turtle graphics module.

The game features two paddles controlled by players and a moving ball that bounces across the screen.
Players score points when the opponent fails to return the ball.

---

## 🧠 Concepts Practiced

* Object-Oriented Programming (OOP)
* Turtle graphics
* Collision detection
* Keyboard event listeners
* Game loops and screen updates
* Working with multiple classes and files
* Score tracking system

---

## 🚀 Technologies Used

* Turtle Module
