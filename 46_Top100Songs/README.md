# 🎵 Spotify Time Machine

This project was completed as part of **Day 46** of the  
**100 Days of Code: The Complete Python Pro Bootcamp** by Angela Yu.

## 📌 Project Overview

This project creates a Spotify playlist based on the **Billboard Hot 100** songs from a date entered by the user. It scrapes the song titles from Billboard and uses the Spotify API to automatically generate a playlist.

---

## 🧠 Concepts Practiced

- Web Scraping
- Working with APIs
- OAuth Authentication
- Beautiful Soup
- Spotify API Integration

---

## 🚀 Technologies Used

- Python
- Requests
- Beautiful Soup
- Spotipy
- Spotify Web API

---

## 💡 What I Learned

Through this project, I learned how to combine web scraping with APIs by extracting song data from a website and using the Spotify API to automate playlist creation.
