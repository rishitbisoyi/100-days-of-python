# 🔢 Guess the Number Game

This project was built as part of **Day 12** of the
**100 Days of Code: The Complete Python Pro Bootcamp** by Angela Yu.

## 📌 Project Overview

Guess the Number is a command-line Python game where the player tries to guess a randomly generated number within a limited number of attempts.

The program provides hints after each guess by telling the player whether the guessed number is:

* Too high
* Too low
* Correct

The game also includes different difficulty levels with varying numbers of attempts.
