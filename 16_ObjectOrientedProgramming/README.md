# ☕ Object-Oriented Coffee Machine

This project was built as part of **Day 16** of the
**100 Days of Code: The Complete Python Pro Bootcamp** by Angela Yu.

## 📌 Project Overview

This project is an improved version of the Digital Coffee Machine built using **Object-Oriented Programming (OOP)** concepts in Python.

The program simulates a coffee vending machine where users can:

* Select coffee drinks
* Insert coins
* Receive drinks if resources and payment are sufficient

Unlike the previous version, this implementation uses classes and objects to organize the code more efficiently and make it easier to manage and scale.
