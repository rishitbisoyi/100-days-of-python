# 🧠 GUI Quiz App

This project was built as part of **Day 34** of the
**100 Days of Code: The Complete Python Pro Bootcamp** by Angela Yu.

## 📌 Project Overview

The GUI Quiz App is a graphical version of the Quiz Game built using **Tkinter** and data retrieved from an online trivia API.

The application presents true/false questions to the user through a graphical interface, provides immediate feedback on answers, and keeps track of the player's score throughout the quiz.

This project combines GUI development with API-based data retrieval to create an interactive quiz experience.
