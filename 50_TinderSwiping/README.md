# ❤️ Auto Tinder Swiping

This project was completed as part of **Day 50** of the  
**100 Days of Code: The Complete Python Pro Bootcamp** by Angela Yu.

## 📌 Project Overview

This project automates the Tinder swiping process using Selenium WebDriver. It logs into a Tinder account, navigates the website, and performs automated swipe actions.

- Browser Automation
- Selenium WebDriver
- User Authentication
- Handling Pop-ups
- Automating User Interactions

---

## 🚀 Technologies Used

- Python
- Selenium WebDriver

## 💡 What I Learned
