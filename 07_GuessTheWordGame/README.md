# 🎯 Guess the Word Game

This project was built as part of **Day 7** of the
**100 Days of Code: The Complete Python Pro Bootcamp** by Angela Yu.

## 📌 Project Overview

Guess the Word is a command-line word guessing game inspired by the classic game **Hangman**.

The player tries to guess a hidden word one letter at a time.
Correct guesses reveal the letters in the word, while incorrect guesses reduce the number of remaining attempts.

The game continues until the player either guesses the full word or runs out of lives.
