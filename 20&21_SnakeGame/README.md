# 🐍 Snake Game

This project was built as part of **Days 20 & 21** of the
**100 Days of Code: The Complete Python Pro Bootcamp** by Angela Yu.

## 📌 Project Overview

The Snake Game is a graphical Python game inspired by the classic arcade game.

The player controls a snake that moves around the screen to collect food and grow longer while avoiding collisions with the walls and its own body.

This project was developed across two days, so both parts are included together in a single folder.

---

## 🧠 Concepts Practiced

* Object-Oriented Programming (OOP)
* Turtle graphics
* Class creation and object interaction
* Keyboard event listeners
* Collision detection
* Game loops and screen updates
* Managing multiple objects dynamically

---

## 🚀 Technologies Used

* Python 3
* Turtle Module

---

## 💡 What I Learned

Through this project, I learned how larger games can be structured using multiple classes and files in Python.

This project also helped me improve my understanding of game loops, collision handling, object movement, and organizing code into reusable components.

## 🎮 Features

* Snake movement controls
* Food spawning system

* Snake growth mechanics
