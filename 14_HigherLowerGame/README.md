# 📈 Higher Lower Game

This project was built as part of **Day 14** of the
**100 Days of Code: The Complete Python Pro Bootcamp** by Angela Yu.

## 📌 Project Overview

The Higher Lower Game is a command-line Python game where the player compares two accounts and guesses which one has a higher follower count.

After each correct guess, the score increases and the game continues with a new comparison.
The game ends when the player makes an incorrect guess.
