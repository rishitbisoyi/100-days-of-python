# 🐢 Turtle Race Game

This project was built as part of **Day 19** of the
**100 Days of Code: The Complete Python Pro Bootcamp** by Angela Yu.

## 📌 Project Overview

The Turtle Race Game is a fun graphical Python project built using the Turtle graphics module.

In this game, multiple turtles race across the screen, and the user places a bet on which turtle will win.
The turtles move forward randomly until one reaches the finish line.
