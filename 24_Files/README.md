# 📂 File Handling & Mail Merge Projects

These projects were completed as part of **Day 24** of the
**100 Days of Code: The Complete Python Pro Bootcamp** by Angela Yu.

## 📌 Project Overview

This day focused on learning how to work with:

* Files
* Directories
* File paths in Python

As part of the practice, I updated the previously built Snake Game by adding a **high score system** and making small improvements to the gameplay experience.

I also created a **Mail Merge** project that automatically generates personalized letters using names from a list and a template file.

---

## 🧠 Concepts Practiced

* Reading and writing files in Python
* Working with directories and file paths
* File handling using `open()`
* String replacement
* Managing external text files
* Improving existing projects
* Basic automation concepts

---

## 🚀 Technologies Used

* Python 3

---

## 💡 What I Learned

Through these projects, I learned how Python can interact with files and automate repetitive tasks.

Updating the Snake Game also helped me understand how existing projects can be improved incrementally by adding new features and refining the user experience.

## 🎮 Features

* High score saving system for Snake Game
* Personalized letter generation
* File reading and writing
* Template-based text replacement
* Improved project structure and organization
