# 🃏 Flash Card App

This project was built as part of **Day 31** of the
**100 Days of Code: The Complete Python Pro Bootcamp** by Angela Yu.

## 📌 Project Overview

The Flash Card App is a GUI-based Python application designed to help users learn and memorize information through digital flashcards.

The application displays a word on one side of a card and automatically flips after a few seconds to reveal its translation. Users can track their learning progress by marking words they already know.
