# 💰 Blind Auction Program

This project was built as part of **Day 9** of the
**100 Days of Code: The Complete Python Pro Bootcamp** by Angela Yu.

## 📌 Project Overview

The Blind Auction Program is a Python application that simulates a secret bidding auction.

Multiple users can enter their names and bid amounts without seeing other participants' bids.
Once all bids are collected, the program determines and displays the highest bidder as the winner.
